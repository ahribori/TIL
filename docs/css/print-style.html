# 도큐먼트 프린트 스타일 정의하기

웹 페이지를 화려하게 꾸며놨는데 프린트하려고 인쇄 미리보기를 띄워보면 와장창 되는 경우가 있음. 이럴 땐, 프린트 할 스타일을 재정의 하면 됨.

예를 들기 위해 div 안에 글을 넣었다.

<div class="print">Hello, World!</div>
1

인쇄할 때, 저 문자들을 빨간색으로 인쇄하고 싶으면 저 문자들에 대한 스타일을 @media print 미디어쿼리로 재정의 하면 된다

.print {
    @media print {
        color: red;
    }
}
