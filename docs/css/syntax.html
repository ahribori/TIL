# 1. 셀렉터 (Selector, 선택자)

스타일을 먹일 HTML 요소들을 선택하기 위해 CSS에서 제공하는 방법. 제목을 빨간색으로 바꾸려면 먼저 제목을 선택해야된다.

p {
    color: red;
}
