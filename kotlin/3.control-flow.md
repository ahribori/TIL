# Control flow

### if-else

-   일반적인 사용 방법

```
var max = a
if (a < b) max = b
```

```
var max: Int

if (a > b) {
    max = a
} else {
    max = b
}
```

-   If 문이 식으로 사용되는 경우 값을 반환
-   If 문이 식으로 사용되는 경우 반드시 else 문도 같이 써줘야함

```
val max = if (a > b) a else b
```

-   If 식의 brancheds 들이 블록을 가질 수 있음
-   블록의 마지막 구문이 반환값

```
val max  = if (a > b) {
    a
} else {
    b
}
```
